Features

NURO Analysis is a web app that uses the data from NURO Connect to identify error points and develop solutions based on specific tests.

Complete Data Access

Authorizes data analysis of boilers from all sites and time periods

Error Diagnostics

Conducts numerous tests to classify equipment errors and their urgencies

Predictive Technology

Highlights warning points where data is trending towards test failure

Solution-Driven

Provides error summaries to pinpoint causes and recommend solutions for test failures

Data Visualization

Generates graphs comparing relevant data and identifying error and warning points

NURO Connect

Collaborates with NURO Connect to cultivate data transparency and error analysis

Analyze Boilers

Use this section as a guide through the boiler analysis process.

Initializing Analysis

To open the Available Boilers page, press the blue "Conduct Analysis" button on the homepage or the "Analyze Boilers" option on the sidebar.

Available Boilers

On the Analyze Boilers page, press the blue "Run Diagnostics" button under the desired boiler to open its Diagnostics page.

Selecting Time Intervals

On the Diagnostics page, fill out the form with a start time, end time, and time interval to select the time period for analysis. The start time must be before the end time, and NURO Connect must have been connected during the time period. Press the blue "Submit" button when the times have been entered.

Error Summary

Error Types
  • Red Errors: These are from tests that have a failure rate of at least 25%. They are important to address as they are the main sources of error for the boiler.
  • Yellow Errors: These are from tests that have a failure rate under 25% but over 0%. They are less urgent but can identify the impact of red errors on the performance of other parts of the boiler and can prevent further equipment error if monitored.

After the analysis is complete, a popup will open showing a summary of the test results. This will give suggestions on how to eliminate future errors, improve efficiency, and find potential solutions. Press the "x" button to close the popup. To open the summary again, press the blue "Open Summary" button. The yellow errors are not shown by default. Press the "Yellow Errors" button to show these messages.

Test Cases

Test Types
  • Red Tests: These tests failed over 25% of the time and are the main source of inefficiency in the boiler.
  • Yellow Tests: These tests failed under 25% of the time but did not pass 100%. They are often the byproduct of other red tests or are trending towards red. Check these tests to prevent future performance issues.
  • Green Tests: These tests had a 0% error.

There are numerous tests to analyze the performance of the boiler. They compare specific datasets to determine if all parts of the boiler are working efficiently.

Generating Graphs

Press the grey "Generate Timeline Graph" button to open a test's graph. Press the grey "Generate Binary Graph" button to show the pass/fail/warning graph that highlights the error and warning statuses. For further help, press the grey question mark button.

Graph Features

Use these tools to easily visualize the data of each test.

Crosshair and Tooltips

Hover over the graph to show a crosshair that highlights the data points for the selected date and time. This will also display a tooltip that shows the labels, values, and colors for the datasets, as well as the test status for that point.

Dataset Popups

Hover the dataset names on the legend to show the popup information for each dataset. Information includes error point conditions, warning point thresholds, and axis usage.

Remove/Add Datasets

On the graph legend, press a dataset's name to hide its points. Press the dataset's name again to add the points back to the graph. This is useful to remove warning points and only show error points, vice versa.

Zoom and Pan

To zoom in, drag the mouse over the desired values or scroll the mouse wheel. To pan across values while zoomed in, press CTRL and use the mouse to drag to the desired points. Press the grey reset button to revert to the original graph.

Mobile Support

NURO Analysis supports mobile devices through some user interface modifications.

Sidebar Navigation

Press the grey button in the top left corner to open the sidebar. The sidebar will help navigate around the site.

Graph Options

Press the grey "Options" button of the desired test to open the graph dropdown. The "Generate Timeline Graph" option opens the test graph. The "Generate Binary Graph" option opens the pass/fail/warning graph to easily see when the data fails the test.

Zoom and Pan

To zoom in/out, pinch the graph. To pan to other data points while zoomed in, drag across the graph.

Hover Features

All hover features are accessible for mobile through touch. Help popups can be opened with one press. The graph crosshair can be displayed by pressing the desired point. Dataset popups and warning thresholds can be shown by double tapping the legend label. One tap would still remove/add the dataset.
